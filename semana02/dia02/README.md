# CRUD de Empresas (Python)

Este script implementa un CRUD simple para empresas (registrar, listar, actualizar, eliminar).

Uso:

- Ejecutar en modo interactivo (menú):

```bash
python 03_ejercicio.py
```

- Ejecutar demo no interactivo (para pruebas rápidas):

```bash
python 03_ejercicio.py --demo
```

Los datos se guardan en `companies.json` en el mismo directorio.

Requisitos: Python 3.6+.
